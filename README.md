# coinigy

Вы хотите протестировать интересную торговую идею и решаете собрать данные. Используя API веб-терминала coinigy.com, вам нужно накопить bid-ask-цены за 10 минут для монет ETH, LTC, ZEC и DASH против BTC, делая снимки каждые 2 секунды. Затем необходимо сделать resample данных и посчитать mid-price с таймфреймом в 20 секунд. По получившимся данным нужно посчитать корреляционную таблицу. В процессе данные должны записываться и храниться в формате pandas.dataframe (если язык – Python).
